# UNESCO AI Ethics Scoring API

This FastAPI-powered application audits AI systems for compliance with UNESCO's AI Ethics Recommendations. It accepts structured submissions, computes weighted compliance scores, and generates multilingual, multi-format reports including PDF, Excel, HTML, and JSON.
